# Malvin TakeIt MD 🤖
**WhatsApp Multi Device Bot by Handsome Tech Zimbabwe**

## Features
- 60+ Economy Commands
- 20 VPN File Commands
- .menu, .ping, and more

## Setup
```bash
npm install
node index.js
```

Scan the QR code with WhatsApp to connect.

## VPN Files
Place your `.ovpn`, `.conf`, or `.txt` files in the `vpnfiles/` folder.
Users can then retrieve them using `.vpnget [filename]`.

## Commands
Type `.menu` in WhatsApp to see all commands.
